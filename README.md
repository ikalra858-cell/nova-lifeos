# 🚀 NOVA – AI Life Operating System

An AI-powered productivity assistant that helps users manage their daily life using natural language.

Built with **React, Flask, SQLite, and Google Gemini 2.5 Flash**, NOVA combines AI chat with productivity tools like tasks, notes, events, goals, memory, voice interaction, and intelligent daily planning.

---

# ✨ Features

- 🤖 AI Chat powered by Google Gemini
- ✅ Task Management
- 📝 Smart Notes
- 📅 Event Scheduler
- 🎯 Goal Tracking
- 🧠 Personal Memory
- 📋 AI Daily Planner
- 🎤 Voice Commands
- 📊 Productivity Dashboard
- 💬 Chat History
- ⚡ Fast React + Flask Architecture

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/ikalra858-cell/nova-lifeos.git
```

---

## Backend

```bash
cd backend

pip install -r requirements.txt

python app.py
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```

# 💡 Motivation

Managing productivity usually requires multiple applications for tasks, notes, calendars, reminders, and AI assistants.

NOVA combines everything into one intelligent Life Operating System that understands natural language and helps users stay organized.
